vue 编译原理

1.我们编写的 vue 代码
<span>{{a}}</span>

2.将代码转化成一个对象（ast 抽象语法树）

<!-- {
    type:"span",
    children:a,
} -->

3.对抽象语法树进行优化（打标记 patchFlag）

<!-- {
    type:"span",
    children:a,
    patchFlag:1
} -->

4.根据树转化后的代码产生一个字符串(js 代码)
createElemntVnode('span',null,[a],1/_TEXT_/)

3.将有 codegenNode 的 ast 抽象语法树转化成 render 函数(字符串拼接)
